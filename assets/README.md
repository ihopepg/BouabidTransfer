# Assets Directory

This directory contains application assets:

## Required Files

- `icon.ico` - Application icon (Windows .ico format)
- `logo.png` - Application logo
- `splash.png` - Splash screen (optional)

## Icon Guidelines

- **Size**: 256x256 pixels minimum
- **Format**: .ico for Windows
- **Design**: Professional, modern, recognizable

## Logo Guidelines

- **Size**: 512x512 pixels or higher
- **Format**: PNG with transparency
- **Usage**: Application branding

## Creating Icons

You can create icons using:
- Online icon generators
- Image editing software (GIMP, Photoshop)
- Icon design tools

For now, placeholder icons can be used during development.
